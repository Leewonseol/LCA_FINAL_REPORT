🎯 핵심 질문: "수많은 여가 관련 변수들 중, 어떤 변수가 행복과 여가만족을 실제로 예측하는가?"

🔍 왜 조건부 퍼뮤테이션 중요도(Conditional Permutation Importance)인가?

⚠️ 문제: 일반 중요도의 한계

여가 관련 변수들(시간, 비용, 만족, 인식 등)은 서로 높은 상관관계를 가진다. 예를 들어, 여가시간이 많은 사람은 여가만족도 높고, 여가비용 지출도 많은 경향이 있다. 이러한 다중공선성(multicollinearity) 상황에서 일반적인 Gini Importance나 Permutation Importance는 상관된 변수들의 중요도를 과대추정하는 문제가 있다.

방법원리한계 / 장점
Gini Importance
(Mean Decrease Impurity)		 트리 분할 시 불순도 감소량의 평균 한계 상관 변수들에 중요도가 분산됨, 범주 수가 많은 변수 과대평가
일반 Permutation
(Mean Decrease Accuracy)		 변수 값을 무작위 셔플 후 성능 하락 측정 한계 상관 변수가 있으면 셔플 효과가 상쇄되어 과소평가
조건부 Permutation
(Conditional Importance)		 다른 모든 변수를 통제한 상태에서 해당 변수만 셔플 장점 변수 간 상관 효과를 제거하고 순수한 고유 기여도만 측정
📝 조건부 퍼뮤테이션의 역할: 예를 들어, '휴일 여가시간(Q13_2_2)'과 '평일 여가시간(Q13_2_1)'은 r=0.60 이상의 상관을 보인다. 일반 퍼뮤테이션에서는 둘 중 하나를 셔플해도 다른 변수가 예측력을 보완하여 중요도가 낮게 나타날 수 있다. 조건부 퍼뮤테이션은 이러한 보완 효과를 통제하여, 각 변수가 다른 변수로 대체 불가능한 고유 정보를 얼마나 가지는지를 정확히 측정한다.

🔍 Key Findings

Finding 1: 시간만으로는 행복하지 않다

Class 4는 휴일시간(70.6%, 1위), 일여가균형(28.1%, 1위)이나 행복 4.31점 최저. 은퇴 후 시간은 풍부하나 경제력·건강 부족으로 행복한 여가 불가능.

Finding 2: 자원의 종합적 충족이 핵심

Class 1(7.67) vs Class 4(4.31) 차이 3.36점(34%). 소득(600만 vs 100만), 건강(64% vs 30%), 만족(52% vs 30%)의 복합 결과.

Finding 3: 직장인의 여가 만족 문제

Class 5는 상용직 35%(유일), 여가만족 1.6%(최저). 시간 확보보다 여가의 질 향상이 필요.

Finding 4: 지역 간 여가 격차

행복 최저 Class 4는 호남권 61%, 읍면 59%. 비수도권 고령 1인가구의 여가 빈곤 문제.
